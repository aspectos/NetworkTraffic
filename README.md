# Compile Dependencies
download and compile FTXUI


```sh
	cd FTXUI
    mkdir build && cd build
    cmake ../
    make
```
Now you've got following libs in build folder
- libcomponent.a
- libdom.a
- libscreen.a

move them to 
> capui/FXTUI/lib/

# Usage
```sh
  sudo ./capui
```
